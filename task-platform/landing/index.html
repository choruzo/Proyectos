Funcionalidades

Todo lo que necesita tu equipo

Diseñado para equipos de desarrollo de software que gestionan trabajo por proyectos y versiones/releases.

📋

Kanban y Lista

Visualiza el trabajo como quieras. Cambia entre vista kanban y tabla en un clic, con filtros por estado, prioridad y etiqueta.

🏷️

Proyectos y Versiones

Organiza las tareas por proyecto y release. El backlog siempre separado de las versiones en desarrollo o entregadas.

👥

Multiusuario con roles

Admin y usuario normal. El admin asigna qué proyectos ve cada persona. Control total sobre accesos sin complicaciones.

📎

Adjuntos hasta 100 MB

Sube capturas, documentos o cualquier archivo directamente a la tarea. Todo almacenado en tu servidor.

🕵️

Historial de auditoría

Cada cambio queda registrado. Quién hizo qué y cuándo, visible en el detalle de cada tarea.

📊

Estadísticas y exportación

Panel de estadísticas global y por proyecto. Exporta todos tus datos a JSON en cualquier momento.

🔒

Datos en tu servidor

Instala con Docker Compose en 10 minutos. Los datos nunca salen de tu infraestructura. RGPD por diseño.

🎨

Modo claro y oscuro

Tema Duna Serena — arena y ámbar. Conmuta entre modo claro y oscuro con un clic.

💾

Vistas guardadas (Presets)

Guarda combinaciones de filtros como presets por proyecto. Accede a tu vista habitual en un clic.

Vista Kanban

El trabajo de un vistazo

Cuatro columnas: Backlog, Por hacer, En desarrollo, Hecho. Cada tarjeta muestra título y prioridad. Filtra en tiempo real sin recargar la página.

  • Filtros por estado, prioridad y etiqueta simultáneos
  • Buscador de texto sobre todas las tareas de la versión
  • Selector de versión/release integrado en la barra
Vista Lista

Tabla cuando necesitas densidad

Título, estado, prioridad, fecha de vencimiento, estimación en horas y etiquetas en una sola fila. Ordena y filtra igual que en kanban.

  • Mismos filtros que kanban, distinto formato
  • Ideal para revisiones de sprint o reuniones de equipo
  • Abre el detalle completo de cada tarea con un clic
Panel de administración

Control total para el administrador

Gestiona usuarios, proyectos, versiones y accesos desde un panel central. Las estadísticas muestran el estado del trabajo en tiempo real.

  • Crear y desactivar usuarios, reset de contraseña
  • Asignar qué proyectos ve cada usuario
  • Gráficos de estado y prioridad por proyecto
  • Exportar todos los datos a JSON
Instalación

En marcha en 10 minutos

Solo necesitas Docker y Docker Compose. Funciona en cualquier servidor Linux, Windows Server o NAS compatible.

# 1. Descarga el proyecto y configura las variables
cp .env.example .env

# 2. Arranca todos los servicios (base de datos, API, web)
docker compose up --build -d

# Ya está. Accede desde tu red local:
# Web → http://tu-servidor:8080
# API → http://tu-servidor:8000/docs
